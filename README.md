## 🚀 **About This Project**

**Ishak's Den** is a cutting-edge, full-stack portfolio website that combines modern web technologies with stunning animations and smooth user experience. Built with the latest React 19 and Next.js 15, it features a complete authentication system, dynamic project management, and a beautiful responsive design.

## � **Getting Started**

### **Prerequisites**

- Node.js 18+
- PostgreSQL database
- Pinata account (for image uploads)

### **Installation**

1. **Clone the repository**

   ```bash
   git clone https://github.com/IshakShekh97/ishak-s-den.git
   cd ishak-s-den
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Set up environment variables**

   ```bash
   cp .env.example .env.local
   ```

   Fill in your environment variables:

   ```env
   # Database
   DATABASE_URL="postgresql://..."

   # Authentication
   BETTER_AUTH_SECRET="your-secret-key"
   BETTER_AUTH_URL="http://localhost:3000"

   # Pinata (for image uploads)
   PINATA_JWT="your-pinata-jwt"
   PINATA_GATEWAY_URL="your-gateway-url"
   ```

4. **Set up the database**

   ```bash
   npx prisma generate
   npx prisma db push
   ```

5. **Run the development server**

   ```bash
   npm run dev
   ```

6. **Open [http://localhost:3000](http://localhost:3000)** in your browser
